# Electsys Utility

上海交通大学教务处实用程序

# 功能

 - [x]  将教务处的课程表信息和你的日历同步。支持本地日历和 CalDAV 日历。

 - [x]  同步考试信息到日历提醒事件中。

 - [ ]  从 [finda-studyroom](https://github.com/yuxiqian/finda-studyroom) 获取课程数据并分析。


# 已知问题

* 日历中包含非 iCloud 的 CalDAV 账户（如：QQ 邮箱同步的日历）时会产生无法写入的错误。
* 当未得到系统日历充分授权时会反应过度。

# 起因
* 受够了 ×× 课程表和 ×× 格子之后，希望可以获得一个方便稳定的方式来给自己课程提示。
> (虽然写起来并不方便，用起来也并不稳定……)
